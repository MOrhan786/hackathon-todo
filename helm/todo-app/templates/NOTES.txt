== Quick Access ==

Get Minikube IP:
  minikube ip

Open Frontend in browser:
  minikube service frontend -n {{ .Values.namespace }}

Open Backend in browser:
  minikube service backend -n {{ .Values.namespace }}

== Check Status ==

  kubectl get pods -n {{ .Values.namespace }}
  kubectl get services -n {{ .Values.namespace }}
