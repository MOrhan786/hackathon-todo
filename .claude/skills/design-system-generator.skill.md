# design-system-generator

## Purpose
Generate design systems including color schemes, typography, spacing, and components

## Responsibilities
- Create consistent color palettes with proper contrast ratios
- Define typography scales and font pairings
- Establish spacing systems and grid structures
- Generate reusable component templates and patterns
- Ensure accessibility compliance in design choices

## Inputs
- Brand guidelines or aesthetic preferences
- Target platform or device constraints
- Accessibility requirements
- Component library specifications

## Outputs
- Complete design system documentation
- Color palette with hex codes and usage guidelines
- Typography scale with font sizes and weights
- Spacing system with measurements and ratios
- Component specifications with variants

## Rules
- NEVER generate inaccessible color combinations
- NEVER violate platform design guidelines
- NEVER create inconsistent spacing or typography
- ALWAYS ensure WCAG compliance for color contrast

## When to use this skill
Use when establishing new design systems, updating brand guidelines, or creating consistent UI frameworks for projects.
