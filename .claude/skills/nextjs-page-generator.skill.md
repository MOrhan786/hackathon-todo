# nextjs-page-generator

## Purpose
Generate beautiful, responsive Next.js pages and components

## Responsibilities
- Create Next.js page components with proper structure
- Implement responsive design patterns
- Add appropriate styling and layout systems
- Generate reusable component templates
- Include accessibility features and best practices

## Inputs
- Page or component requirements
- Design system specifications
- Layout and content structure
- Responsive design constraints

## Outputs
- Next.js page or component files
- Associated CSS modules or styling
- Component composition patterns
- Responsive behavior implementations
- Accessibility markup and attributes

## Rules
- NEVER create inaccessible user interfaces
- NEVER ignore responsive design principles
- NEVER generate inefficient or bloated components
- ALWAYS follow Next.js best practices

## When to use this skill
Use when creating UI components, developing pages, or implementing frontend features in Next.js applications.
